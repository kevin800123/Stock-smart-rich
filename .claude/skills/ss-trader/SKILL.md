# SS 操盤手方法論（源自 LINE 對話實錄蒸餾）

> 本 skill 蒸餾自使用者提供的 Ss LINE 對話紀錄（約 2022 年空頭期間）。核心世界觀：
> **「任何線形都是主力畫出來的，籌碼（量、主力分點）最可靠。」**
> 所有分析輸出必須附註：非投資建議，僅為規則化的經驗法則對照。

## 0. 每日例行檢查（依此順序）

**盤前**：夜盤（對應美股，與日盤關聯不大）、台幣匯率、日韓股是否跌破前日低點、美股三大（小那 vs 小道瓊 vs 費半的相對強弱）。
**盤中**：台幣匯率（急升→大盤不易跌；急貶→資金流出）、高價股（高價股漲停＝主力作多訊號）、權值股關鍵價位是否有大單（主力守護處）。
**盤後**：道瓊、小那、融資融券、期權（P/C 比）、主力分點進出。

本專案可直接取數的 API：`/api/dashboard`（大盤/法人/融資券/國際）、`/api/os-futures`（小道瓊/小那/費半）、`/api/tx/volume-sessions`（台指期日盤/夜盤量）、`/api/index/kline?symbol=tx`（波浪標記）、`/api/traders/ss`（本方法論的量化檢核表，多操盤手註冊表的一員）。

## 1. 大盤判讀

### 底部訊號（多項同時成立才可信）
- **融資維持率 13X%**：「除非有重大事件，不然融資維持率 13X% 算低了，可以抄底」；融資斷頭才會全力反彈。
- **融資跌幅 > 大盤跌幅**：籌碼清洗，底部。
- **散戶極度悲觀**：「散戶都怕的時候，那就是很好的底部」；VIX 極度恐慌＝散戶拋、大戶接；「行情總在全面毀滅中出現」。
- **利空測試的底部才會穩**：「底部是利空測試出來的」；突發性利空會再殺一次，別絕望。
- **量能**：底部出量＝多軍力守；底部沒爆量表示並未太壞；加速趕底時量能放大；「級殺才有好買點」。
- **末跌段特徵**：連最賺錢、本益比最低的公司都出現系統性跌停＝加速趕底。
- **抄底方式**：抄底大盤可買大盤 ETF 風險最小；個股會領先大盤先走（大盤是落後指標）；最先大跌的股票會先領漲；環球晶常有指標帶頭意義。

### 頭部/賣出訊號
- **高點爆量要跑**；主力出貨前一定拉高；大漲時注意分點主力是否出貨。
- **急漲通常是賣點，不要追高**；漲多就會拉回，逢高要賣。
- **三次過不了前高**＝竭盡買盤遇壓力。
- 「前面有山你還會直直去撞嗎？我們出獲利放口袋。」

### 資金流向與匯率
- **小那漲幅 > 小道瓊**＝資金在科技股（利台股電子）；**費半又強於小那**＝更明確。美國漲金融→台股跟著漲金融。
- **台幣急升**→大盤就不會跌（熱錢流入）；**匯率跌（台幣升）**→大盤漲機會高。
- 判別崩盤 vs 洗盤：「最後一盤下殺叫洗盤；如果是崩盤，台幣會狂貶。」
- 美股率先站上 5 日線可帶動台股；美股 3 天沒破底＝穩。

### 波浪架構（Ss 高度依賴）
- 完整五浪走完會反彈；「反彈時機：籌碼＋技術面＋波浪完整五浪」。
- A 波修正要有爆大量才有明確方向；大 B 波信心不足、量能不足、區間震盪；**B 波記得跑（多頭逃命線）**；B 波假破底是主力常用洗法；**C 波必來但會先彈**，C 波＝主跌段破壞力最強。
- 時間比例：大 A 走多久、大 B 就會多久（普通抓 1:1，保守 1:0.5）。
- 幅度推算：高點−低點＝波長，波長×1.382（黃金切割）。
- 鐵律（違反即不成立）：②浪不破①浪起點；③浪絕不最短；④浪不重疊①浪價區。

## 2. 台指期操作紀律

- **設好停損，不要凹單**；「大轉小賠，要懂得停損」——賺大賠小是唯一活路。
- **結算日**：站在空方看多單、不想輸太多會拉近——結算週用籌碼＋技術分析判月結算方向；逆價差主力會想辦法讓它慢慢收斂。
- 夜盤對應美股，與日盤關係不大；台股看匯率。
- 震盪盤打法：拉回接多、跳空開高用「把擋的」思維來回操作；趨勢往上時急殺就是接。

## 3. 選股方法

- **首選條件「季季高」**（營收季季高）：「大盤拉回，個股逆著走。選股條件：季季高」。
- **三率三升＋看分點選股**：毛利率/營益率/淨利率齊升，配合波段主力分點進場。
- **高殖利率＋營收季季高**（升息縮表年代的防禦選股）。
- **錯殺股**：跌深過程中「內資集買值有明顯買盤」＝主力接錯殺籌碼。
- **獲利爆發**：「股價會反應：第一季賺贏去年整年」；Q2 營收比 Q1 雙位數成長→上半年獲利可能超過去年全年。
- **題材節奏**：「飆在題材、盤在營收、跌在盈餘」；盤面有主流股就不用擔心崩盤；買有未來性的股票。
- **K 線觸發**：一紅吃三黑＝轉強可試單；低位跳空十字星＝底部訊號；均線糾結＋趨勢往上（右側交易抓起漲）；KD 低檔 20＋站上 5 日線、季線有撐。
- **量**：「量是唯一不會騙的東西」；買點＝量價穩定＋大盤強＋站上均價線＋過開盤價。
- **避開**：增資股（拉高就是出貨）、除權息股不要貪（融資 <130% 未自砍會被強制補到 166%）；投信 6 月底/12 月底做帳行情，投信持股小心被賣。
- **主力視角**：飆股要飆前主力一定洗到你怕；籌碼越乾淨他收得越多；有時破線破底反而是飆股；主力會在 CB 先做手腳。

## 4. 部位管理（金字塔加碼）

1. **試單**有獲利 → 加碼到基本持股。
2. **基本持股**有獲利 → 再加碼到單一個股最大部位。
3. 跌回加碼部位 → 開始減碼，**一定讓自己有獲利帶走**。
- 股市要玩閒錢；波段交易者不用短線看待；不要見漲說漲、見跌說跌。

## 5. 心法（分析時引用以校準情緒）

- 「不要被市場牽著走」「別用情緒做股票，那只會讓你追高殺低」。
- 「培養自己的一套方式，否則永遠被牽著鼻子走」。
- 「在市場生存比什麼都重要；跌倒了還能站起來才是榮耀」。
- 「超前部署，要有預測的能力」；新聞參考就好——你看得懂，主力看不懂嗎？

## 使用方式

被要求分析時：
1. 先打上述 API 取得當日數據（或請使用者提供）。
2. 依「0. 每日例行檢查」順序過一遍，逐項標註 多/空/中性/資料不足。
3. 大盤結論須同時給出：波浪位置假設、量能判讀、籌碼（融資/法人/散戶多空比）三個維度，不可單一維度下結論。
4. 選股僅列「符合規則的候選＋各自命中哪幾條」，不給買賣指令；結尾附免責：規則化經驗法則對照，非投資建議。
